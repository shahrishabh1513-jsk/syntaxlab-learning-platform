📘 HTML Basics

HTML (HyperText Markup Language) uses "tags" to structure content. The first step is understanding the basic skeleton: !DOCTYPE html, html, head, and body tags.

<!DOCTYPE html>
<html>
<head><title>My Page</title></head>
<body><h1>Hello World!</h1></body>
</html>

🏷️ Elements & Tags

Tags like <p> for paragraphs, <h1> to <h6> for headings. Most tags have an opening and closing version.

<h1>Main Title</h1>
<p>This is a paragraph with <strong>bold</strong> text.</p>

🔗 Attributes & Links

Links use the <a> tag with href attribute. Attributes give extra info to elements.

<a href="https://www.example.com" target="_blank">Visit Example</a>
<img src="photo.jpg" alt="description">

🖼️ Images & Multimedia

Embed images using <img> , videos using <video> tag. Always add alt text for accessibility.

<img src="logo.png" alt="SyntaxLab Logo" width="200">
<video controls><source src="intro.mp4" type="video/mp4"></video>

📊 HTML Tables

Tables allow data in rows and columns using <table>, <tr>, <td>.

<table border="1">
  <tr><th>Language</th><th>Difficulty</th></tr>
  <tr><td>HTML</td><td>Easy</td></tr>
</table>

📝 Forms & Input

Collect user data with forms: text fields, buttons, checkboxes.

<form action="/submit">
  Name: <input type="text" name="name">
  <input type="submit" value="Send">
</form>

✨ Semantic HTML5

Use <header>, <nav>, <article>, <footer> for better SEO and accessibility.

<article>
  <header><h2>Blog Post</h2></header>
  <p>Content here...</p>
  <footer>Author: SyntaxLab</footer>
</article>
