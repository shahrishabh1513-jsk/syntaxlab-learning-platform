🎨 CSS Syntax & Selectors

CSS consists of selectors and declaration blocks. Selectors target HTML elements to style them.

/* Element selector */
p {
  color: blue;
  font-size: 16px;
}

/* Class selector */
.btn {
  background-color: green;
}

/* ID selector */
#header {
  text-align: center;
}

🎨 Colors & Backgrounds

Use color names, hex codes, RGB, or HSL. Background properties add depth to designs.

body {
  background: linear-gradient(135deg, #667eea, #764ba2);
  color: #ffffff;
}

.card {
  background-color: rgba(255, 255, 255, 0.1);
  backdrop-filter: blur(10px);
}

✍️ Typography & Fonts

Control text appearance with font-family, size, weight, and spacing.

h1 {
  font-family: 'Inter', sans-serif;
  font-size: 3rem;
  font-weight: 800;
  letter-spacing: -0.02em;
  text-transform: uppercase;
}

📦 Box Model

Every element is a box with margin, border, padding, and content.

.box {
  width: 300px;
  padding: 20px;
  border: 2px solid #667eea;
  margin: 10px auto;
  border-radius: 12px;
}

💪 Flexbox Layout

One-dimensional layout model for distributing space and aligning items.

.container {
  display: flex;
  justify-content: space-between;
  align-items: center;
  gap: 20px;
  flex-wrap: wrap;
}

📐 CSS Grid

Two-dimensional grid-based layout system for complex designs.

.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: 20px;
}

.item {
  background: rgba(255,255,255,0.1);
  padding: 20px;
}

📱 Responsive Design

Use media queries to adapt layout for different screen sizes.

/* Mobile first design */
@media (max-width: 768px) {
  .grid-container {
    grid-template-columns: 1fr;
  }
  
  h1 {
    font-size: 1.8rem;
  }
}

✨ Animations & Transitions

Create smooth interactive effects with transitions and keyframe animations.

.button {
  transition: transform 0.3s ease, background 0.3s;
}

.button:hover {
  transform: translateY(-5px);
  background: #764ba2;
}

@keyframes fadeIn {
  from { opacity: 0; }
  to { opacity: 1; }
}
