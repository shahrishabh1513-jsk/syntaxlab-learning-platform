🐍 Python Basics & Syntax

Python uses indentation instead of braces. The print() function outputs text to the console.

# This is a comment
print("Hello, Python!")

# Variables don't need explicit declaration
name = "SyntaxLab"
age = 5

📦 Variables & Data Types

Python supports integers, floats, strings, booleans, and more. Type is determined automatically.

# Numeric types
age = 25                # int
price = 19.99           # float

# String
language = "Python"

# Boolean
is_awesome = True

# Check type
print(type(age))        # <class 'int'>

⚖️ Control Flow (if/else)

Conditional statements control program execution flow.

score = 85

if score >= 90:
    grade = "A"
elif score >= 70:
    grade = "B"
else:
    grade = "C"

print(f"Your grade is {grade}")

🔄 Loops (for, while)

Iterate over sequences or repeat code blocks.

# For loop
fruits = ["apple", "banana", "orange"]
for fruit in fruits:
    print(fruit)

# Range loop
for i in range(5):
    print(f"Count: {i}")

# While loop
count = 0
while count < 3:
    print(f"While loop: {count}")
    count += 1

⚙️ Functions & Modules

Functions organize reusable code. Modules import external functionality.

# Define a function
def greet(name):
    return f"Hello, {name}!"

# Call the function
message = greet("Python Learner")
print(message)

# Default parameters
def multiply(a, b=2):
    return a * b

# Import module
import math
print(math.sqrt(16))  # 4.0

📊 Lists & Dictionaries

Lists store ordered collections. Dictionaries store key-value pairs.

# Lists
colors = ["red", "green", "blue"]
colors.append("yellow")
colors.remove("green")

# List comprehension
squares = [x**2 for x in range(5)]

# Dictionaries
person = {
    "name": "John",
    "age": 30,
    "city": "New York"
}
print(person["name"])

# Dictionary methods
person["email"] = "john@example.com"

📄 File Handling

Read from and write to files using built-in functions.

# Writing to file
with open("data.txt", "w") as file:
    file.write("Hello Python!\n")
    file.write("Learning file handling")

# Reading from file
with open("data.txt", "r") as file:
    content = file.read()
    print(content)

# Reading line by line
with open("data.txt", "r") as file:
    for line in file:
        print(line.strip())

🏗️ Object-Oriented Programming

Classes and objects enable structured, reusable code.

class Student:
    def __init__(self, name, grade):
        self.name = name
        self.grade = grade
    
    def display(self):
        return f"{self.name} scored {self.grade}"
    
    @staticmethod
    def school_name():
        return "SyntaxLab Academy"

# Create object
student1 = Student("Alice", "A")
print(student1.display())

# Inheritance
class GraduateStudent(Student):
    def __init__(self, name, grade, thesis):
        super().__init__(name, grade)
        self.thesis = thesis
