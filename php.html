🐘 PHP Syntax & Basics

PHP code is executed on the server. It's embedded in HTML using <?php ?> tags.

<?php
// This is a PHP comment
echo "Hello, World!";
?>

📊 Variables & Data Types

Variables start with $ sign. Supports strings, integers, floats, booleans, arrays, and objects.

<?php
$name = "SyntaxLab";
$age = 5;
$price = 29.99;
$isActive = true;

echo "Name: " . $name;
?>

⚖️ Conditional Statements

Control program flow with if, else, elseif, and switch statements.

<?php
$score = 85;

if ($score >= 90) {
    echo "Grade A";
} elseif ($score >= 70) {
    echo "Grade B";
} else {
    echo "Need improvement";
}
?>

🔄 Loops in PHP

Repeat code blocks with for, while, foreach loops.

<?php
// for loop
for ($i = 0; $i < 5; $i++) {
    echo $i . "<br>";
}

// foreach for arrays
$colors = ["red", "green", "blue"];
foreach ($colors as $color) {
    echo $color . "<br>";
}
?>

⚙️ Functions & Arrays

Create reusable functions and work with indexed/associative arrays.

<?php
function greet($name) {
    return "Hello, $name!";
}

// Array functions
$fruits = ["apple", "banana", "orange"];
array_push($fruits, "grape");
$count = count($fruits);

$person = ["name" => "John", "age" => 30];
echo $person['name'];
?>

📝 Form Handling (GET/POST)

Retrieve data from HTML forms using $_GET or $_POST superglobals.

<form method="POST" action="process.php">
  <input type="text" name="username">
  <input type="submit">
</form>

<?php
if ($_SERVER["REQUEST_METHOD"] == "POST") {
    $username = $_POST['username'];
    echo "Welcome, $username!";
}
?>

🗄️ Working with Databases

Connect to MySQL database using PDO or MySQLi for data operations.

<?php
$pdo = new PDO("mysql:host=localhost;dbname=test", "user", "pass");

$stmt = $pdo->prepare("SELECT * FROM users WHERE id = ?");
$stmt->execute([$id]);
$user = $stmt->fetch();

// Insert data
$sql = "INSERT INTO users (name) VALUES (:name)";
$stmt = $pdo->prepare($sql);
$stmt->execute(['name' => $name]);
?>

🔐 Sessions & Cookies

Maintain user state across pages using sessions or store data in cookies.

<?php
session_start();
$_SESSION['user_id'] = 123;

// Set cookie
setcookie("theme", "dark", time() + 86400, "/");

// Access session data
if (isset($_SESSION['user_id'])) {
    echo "User is logged in";
}
?>
