🔧 C Syntax & Structure

C programs start with headers and a main() function. Every statement ends with a semicolon.

#include <stdio.h>

int main() {
    printf("Hello, World!");
    return 0;
}

📊 Variables & Data Types

Common data types: int, float, double, char. Variables must be declared before use.

int age = 25;
float price = 19.99;
char grade = 'A';
double pi = 3.14159;

🧮 Operators

Arithmetic, relational, logical, and bitwise operators perform operations on data.

int sum = 10 + 5;
int isEqual = (a == b);
int result = (a > b) && (c < d);

⚖️ Control Flow

Use if-else and switch statements for conditional execution.

if (score >= 60) {
    printf("Pass");
} else {
    printf("Fail");
}

switch(day) {
    case 1: printf("Monday"); break;
    case 2: printf("Tuesday"); break;
    default: printf("Other day");
}

🔄 Loops

Repeat code blocks with for, while, and do-while loops.

// for loop
for(int i = 0; i < 5; i++) {
    printf("%d ", i);
}

// while loop
int i = 0;
while(i < 5) {
    printf("%d ", i);
    i++;
}

⚙️ Functions

Functions help organize code into reusable blocks. They can return values and take parameters.

int add(int a, int b) {
    return a + b;
}

int main() {
    int result = add(10, 20);
    printf("Sum: %d", result);
    return 0;
}

📚 Arrays & Strings

Arrays store multiple elements of same type. Strings are character arrays.

int numbers[5] = {1, 2, 3, 4, 5};
char name[] = "SyntaxLab";

// Access array elements
for(int i = 0; i < 5; i++) {
    printf("%d ", numbers[i]);
}

📍 Pointers

Pointers store memory addresses. They enable dynamic memory management.

int x = 10;
int *ptr = &x;

printf("Value: %d\n", *ptr);  // Dereferencing
printf("Address: %p\n", ptr);

// Changing value through pointer
*ptr = 20;
