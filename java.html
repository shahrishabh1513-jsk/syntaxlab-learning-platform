☕ Java Syntax & Setup

Java programs run on JVM. Every application needs a main method.

public class Main {
    public static void main(String[] args) {
        System.out.println("Hello, Java!");
    }
}

📦 Variables & Data Types

Java is strongly typed. Primitive and reference types.

int age = 25;
double price = 19.99;
char grade = 'A';
boolean isActive = true;
String name = "SyntaxLab";

🏗️ OOP Concepts (Classes)

Classes blueprints for objects. Encapsulation with private fields and public methods.

public class Student {
    private String name;
    private int age;
    
    public Student(String name, int age) {
        this.name = name;
        this.age = age;
    }
    
    public void display() {
        System.out.println(name + " is " + age);
    }
}

Student s1 = new Student("John", 20);
s1.display();

🌳 Inheritance & Interfaces

Extend classes with 'extends' and implement contracts with 'implements'.

interface Drawable {
    void draw();
}

class Circle implements Drawable {
    public void draw() {
        System.out.println("Drawing circle");
    }
}

class Vehicle {
    void start() {
        System.out.println("Vehicle started");
    }
}

⚠️ Exception Handling

Handle errors using try-catch-finally blocks.

try {
    int result = 10 / 0;
} catch (ArithmeticException e) {
    System.out.println("Cannot divide by zero!");
} finally {
    System.out.println("This always executes");
}

HashMap<String, Integer> map = new HashMap<>();

🧵 Multithreading

Run multiple tasks concurrently using Thread class or Runnable interface.

class MyThread extends Thread {
    public void run() {
        System.out.println("Thread running");
    }
}

MyThread t1 = new MyThread();
t1.start();

// Using Runnable
Runnable task = () -> {
    System.out.println("Runnable task");
};
new Thread(task).start();

📄 File Handling

Read and write files using FileReader, FileWriter, and Scanner.

import java.io.*;

// Writing to file
FileWriter writer = new FileWriter("data.txt");
writer.write("Hello Java!");
writer.close();

// Reading from file
BufferedReader reader = new BufferedReader(new FileReader("data.txt"));
String line = reader.readLine();
System.out.println(line);
