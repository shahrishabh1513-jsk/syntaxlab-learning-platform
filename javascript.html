📦 Variables & Data Types

JavaScript variables store data values. Use let, const, or var. Data types include strings, numbers, booleans, and more.

let name = "SyntaxLab";
const year = 2026;
var isAwesome = true;

let count = 42;
let price = 19.99;
let languages = ["HTML", "CSS", "JS"];

🧮 Operators & Expressions

Arithmetic, comparison, and logical operators help manipulate data.

let sum = 10 + 5;
let isEqual = (10 === 10);
let isAdult = (age >= 18) ? "Yes" : "No";

// Logical operators
if (isLoggedIn && hasPermission) {
  console.log("Access granted");
}

🔄 Control Flow

Control the flow of execution with conditionals and loops.

// if-else statement
if (score >= 90) {
  grade = 'A';
} else if (score >= 70) {
  grade = 'B';
}

// for loop
for (let i = 0; i < 5; i++) {
  console.log("Count: " + i);
}

// while loop
let counter = 0;
while (counter < 3) {
  counter++;
}

⚙️ Functions & Scope

Functions are reusable blocks of code. Scope determines variable accessibility.

function greet(name) {
  return `Hello, ${name}!`;
}

const add = (a, b) => a + b;

// Scope example
let globalVar = "I'm global";

function showVar() {
  let localVar = "I'm local";
  console.log(globalVar); // Accessible
}

📊 Arrays & Objects

Arrays store ordered lists. Objects store key-value pairs for structured data.

// Array methods
let fruits = ["apple", "banana", "orange"];
fruits.push("grape");
fruits.forEach(fruit => console.log(fruit));

// Objects
let user = {
  name: "SyntaxMaster",
  age: 25,
  skills: ["JS", "React", "Node"]
};

console.log(user.name);

🎯 DOM Manipulation

Interact with HTML elements dynamically using the Document Object Model.

// Selecting elements
const heading = document.querySelector('h1');
heading.textContent = "Hello JavaScript!";

// Creating elements
const newDiv = document.createElement('div');
newDiv.innerHTML = '<p>Dynamic content</p>';
document.body.appendChild(newDiv);

🎪 Events & Callbacks

Respond to user actions like clicks, typing, and mouse movements.

const button = document.getElementById('myButton');
button.addEventListener('click', () => {
  alert('Button clicked!');
});

// Input event
input.addEventListener('input', (e) => {
  console.log(e.target.value);
});

☁️ Async JavaScript

Handle asynchronous operations with Promises and async/await.

// Promise example
fetch('https://api.example.com/data')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error(error));

// Async/Await
async function getData() {
  try {
    const response = await fetch(url);
    const data = await response.json();
    return data;
  } catch (error) {
    console.log(error);
  }
}
